In order to run on Electron.

Open up a Node.js command prompt, and navigate to the root folder of the project.

If you don't already have dependencies installed, do 'npm install' in the command prompt in order to install them.

To install electron globally, type 'npm install -g electron'

To run on electron you will type 'electron .' or 'npm start'

this will compile the main.js file and create an application with js, html, and css files.
